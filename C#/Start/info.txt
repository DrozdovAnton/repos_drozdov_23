2.1. Объявление (= создание) перемененной:
	тип_данных имя_переменной;


2.2. Инициализация переменной:
	имя_переменной = значение_переменной;


2.3. Объявление и инициализация переменной в один момент:
	тип_данных имя_переменной = значение_переменной;


2.4. Для такого типа данных, как char - используем одинарнык ковычки 't' для инициализации символа t,
а для string - используем двойные ковычки "text" для инициализации строки text.


3.1. Ввод данных в консоль - осуществляется с помощью using System (по аналогии с вывовод в консоль -
- Console.WriteLine(параметр_метода)):
	using Sytem;
	...
	string data; // определяем переменную data
	data = Console.ReadLine(); // метод ReadLine не принимает никакие параметры! 
	// ReadLine() - получение в консоле данных с типом string
	Console.WriteLine(); // Вывод в консоле - строка, полученная от пользователя


3.2. Операция конкатенация - склейка строк:
	Console.WriteLine("Привет "+"мир"); // автоматически пробелы не расставляются, 
	// поэтому не забываем про них!


4.1. В пункте 3.1., строка 38 указывается, что ReadLine() работает с типом данных string.
Поэтому разберемся с конвертацией, например, string -> int:
	System.Convert.ToInt32(параметр_метода); // где параметр_метода - это string
